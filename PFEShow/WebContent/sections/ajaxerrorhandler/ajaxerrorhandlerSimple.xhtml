Default behaviour uses English texts, but most of the applications are created in the local language and for that reason all the texts must be localized. Component pe:ajaxErrorHandler enables showing errors in the given language with the help of defining localized texts.

Attributes title, body, button enable defining the texts for the correct displaying in a modal window. In the texts you can use the following variables.

  • {error-name} â€“ complete name of the exception/error, e.g javax.faces.application.ViewExpiredException
  • {error-message} - Message of exception/error
  • {error-stacktrace} - stacktrace of exception/error (work in extended mode)
  • {error-hostname} - hostname of server, where exception is catched. This is good information in clusters solutions (work in extended mode)

Default settings is: <pe:ajaxErrorHandler title="{error-name}" body="{error-message}" button="Reload" />
