In case if the pe:ajaxErrorHandler would have defined childs compontents and they are not facet, you are defining the content of the window. You can use HTML,JSF components but we strongly not recommend defining h:form or ajax events, that could cause more types of errors.

In custom content is good to define content with class 'ui-dialog-titlebar' and 'ui-dialog-content', where:
  • ui-dialog-titlebar - handle area for change mouse icon for draggable
  • ui-dialog-content - is area, where mouse event for draggable is disabled

Note: work in extended mode only
