Default behaviour of pe:ajaxErrorHandler is to display exception with RELOAD button. There is not neccessary to render this component for all exceptions in the same manner.

For example: Description of ViewExpiredException could be: "Due to inactivity on page please push RELOAD to continue."

For other exceptions it's appropriate to show following: name of exception, message of exception, stacktrace, server name and information for user how to continue.

Attribute TYPE determines behaviour of component - if it is not set AjaxErrorHandler has default settings, otherwise TYPE atribute has value of full class name of exception and for each exception type defines specific properties.
