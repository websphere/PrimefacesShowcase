Global tooltip uses title values of JSF components. Global tooltips are normally applied to all components on a page. But this behavior can be done much smarter. Global tooltips can be limited to specified by "for" / "forSelector" components. The example below shows a case when tooltips only appear for components which didn't pass validation (in this case required inputs). This is a quite nice replacement for regular error messages. ${showcase:getFileContent('/sections/tooltip/example-globalLimitedTooltips.xhtml')}
