In EL 2.2 or older, it's not possible to use constants or any other static fields/methods in an EL expression.
As it is not really "cool" to create beans with getter/setter for each constants class, we provide an utils tag which allows to import constant fields in a page.

The constants can be accessed via the name of the class (default setting) or via a custom name (var attribtue). ${showcase:getFileContent('/sections/utils/example-importConstants.xhtml')} public final class Constants { public static final String LIBRARY = "primefaces-extensions"; public static final String LIBRARY_UNCOMPRESSED = "primefaces-extensions-uncompressed"; public static final String DELIVER_UNCOMPRESSED_RESOURCES_INIT_PARAM = "org.primefaces.extensions.DELIVER_UNCOMPRESSED_RESOURCES"; public static final String EXTENSION_CSS = ".css"; public static final String EXTENSION_JS = ".js"; /** * Avoid instantiation. */ private Constants() { } }
