In EL 2.2 or older, it's not possible to use enum constants or any other static fields/methods in an EL expression.
As it is not really "cool" to create beans with getter/setter for each enum constants class, we provide an utils tag which allows to import enum values in a page.

The enum values can be accessed via the name of the class (default setting) or via a custom name (var attribtue).
It also possible to get all enum values of the class with the "ALL_VALUES" suffix or a custom prefix via the "allSuffix" attribute. ${showcase:getFileContent('/sections/utils/example-importEnum.xhtml')} public enum ProjectStage { Development, UnitTest, SystemTest, Production; }
