CodeMirror is a JavaScript library that can be used to create a relatively pleasant editor interface for code-like content â€• computer programs, HTML markup, and similar. If a mode has been written for the language you are editing, the code will be coloured, and the editor will optionally help you with indentation.

CodeMirror also provides CSS for some modes. If you want to inclue this, they will aply to ALL CodeMirror instances on the page.

Following CSS are available:
  • codemirror/mode/spec.css
  • codemirror/mode/tiddlywiki.css
  • codemirror/mode/tiki.css
You can include this CSS via:
<h:outputStylesheet library="primefaces-extensions" name="codemirror/mode/tiki.css" />
