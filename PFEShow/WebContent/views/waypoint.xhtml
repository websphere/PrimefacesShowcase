Waypoint makes it easy to execute a custom logic whenever you scroll to an element. Waypoints make a solid base for modern UI patterns that depend on a user's scroll position on a page. Application areas are countless, explore our use cases self!

Using: Waypoint component can be used either with pe:javascript or p:ajax. There is only one event "reached". A callback function called when the user scrolls past the element. In the callback function you can access the scrolling direction ("down" or "up") and the current waypoint element as a DOM object.
